Python中一次读取多个值

2017-02-12T02:52:39Z   Python   0 评论  

Python 2里面读取输入的函数是raw_input()， Python 3的是input()，读入一个值后回车读取输入就退出了，想要读取一次读取多个输入，可以像下面这样：

a, b = raw_input().split()

输出的是字符串，要想读取的是数值，可以稍微改一下，像这样：

a, b = map(int, raw_input().split())
